# 🤖 Jarvis AI Assistant

A voice-controlled AI assistant built using Python that can perform tasks like opening websites, playing music, and responding intelligently using AI or offline mode.

---

## ✨ Features

* 🎙️ Voice recognition (wake word: "Jarvis")
* 🔊 Text-to-speech response
* 🌐 Opens websites (Google, YouTube, etc.)
* 🎵 Plays music
* 🤖 AI-powered responses (when online)
* ⚡ Offline mode fallback (basic smart replies)

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/jarvis.git
cd jarvis
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Create `.env` file

```env
OPENAI_API_KEY=your_api_key_here
```

---

## ▶️ Run the project

```bash
python main.py
```
